# 🎯 Atalhos de Teclado - eControle

## Atalhos Globais

| Atalho | Função | Disponibilidade |
|--------|--------|-----------------|
| **Ctrl/Cmd + K** | Focar na barra de pesquisa global | Sempre |
| **Alt + ↑** (Seta para Cima) | Rolar até o topo da aba aberta | Quando em uma aba |

## Atalhos de Navegação entre Abas

| Atalho | Aba | 
|--------|-----|
| **Alt + 1** | 📊 Painel |
| **Alt + 2** | 🏢 Empresas |
| **Alt + 3** | 🔐 Certificados |
| **Alt + 4** | 📄 Licenças |
| **Alt + 5** | 💰 Taxas |
| **Alt + 6** | ✅ Processos |
| **Alt + 7** | 🔗 Úteis |

## Comportamento

### Pesquisa Global (Ctrl/Cmd + K)
- Foca automaticamente no input de pesquisa
- Funciona em qualquer aba
- Limpo e intuitivo para navegação rápida

### Voltar ao Topo (Alt + ↑)
- Rola suavemente até o topo da aba atual
- Funciona em qualquer aba aberta
- Atalho ergonômico com a seta do teclado
- Comportamento smooth (animado)

### Navegar entre Abas (Alt + 1-7)
- Muda instantaneamente para a aba selecionada
- Não interfere com campos de entrada (input, textarea, combobox)
- Foca no botão da aba para melhor acessibilidade

## Notas Importantes

⚠️ **Os atalhos Alt+número e Alt+↑ NÃO funcionam** quando você está digitando em:
- Campos de texto (input)
- Áreas de texto (textarea)  
- Selects (combobox)
- Campos editáveis

Isso evita interferência com a digitação normal.
